shortStack

a simple table pagination / search / sort plugin that doesn't mess with your DOM

View the Project on GitHub rockerest/shortStack

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

shortStack is a simple jQuery plugin that provides instant search and pagination without messing with your DOM very much (It just adds a control bar right after your table).

Make sure your table is well formatted, with at least a 

<tbody></tbody>

Using shortStack is as easy as calling the constructor

$( "table" ).shortStack();
